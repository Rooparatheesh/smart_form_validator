# smart_form_validator

A reusable Flutter package providing modular and extensible form validation utilities including:

- Required field validation
- Email validation
- Mobile number validation
- Password validation
- Date validation

This package helps simplify and standardize form validation logic in Flutter applications.

---

## Features

✔ Required field validator  
✔ Email format validator  
✔ Mobile number validator  
✔ Password strength validator  
✔ Date range validator  
✔ Clean and modular structure  

---
